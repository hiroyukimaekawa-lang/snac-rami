# スナック蘭美 公式ホームページ

## デザイン・構成の解説

### トーン・カラー設計
- **ベースカラー**: 日本の伝統色「梅鼠（うめねずみ）」#9E7A7A を基調に使用
- **背景**: 白（#ffffff）と薄いグレー（#f8f8f8）で清潔感を演出
- **文字色**: 濃いグレー（#222222）から黒で可読性を確保
- **色数**: 最小限に抑え、派手な色やネオンカラーは使用していません
- **写真**: 彩度とコントラストを抑え、落ち着いた印象に調整

### レイアウト・余白
- 参考サイト（https://www.shizuka-kagurazaka.com/）の余白の取り方と落ち着いたトーンを参考に設計
- セクション間の余白を広めに取り、大人向けの静けさを表現
- 和モダン × 夜の雰囲気を意識したデザイン

### 情報設計
- **電話問い合わせ導線を最優先**: ヘッダー、HERO、INFORMATION、ACCESS、フッター、モバイル固定ボタンで多重に配置
- **一見さんでも入りやすい安心感**: CONCEPTセクションでアットホームさを強調
- **20歳未満入室不可**: 控えめに注意書きを表示

### ページ構成

1. **HERO**
   - 落ち着いた写真を背景に
   - 店名「スナック蘭美」を大きく表示
   - キャッチコピー「静かに語らう、大人のくつろぎ」
   - 電話予約CTAボタン

2. **CONCEPT**
   - お店の想いとアットホームさを説明
   - 20歳未満入室不可の注意書き（控えめに）

3. **INFORMATION**
   - 営業時間・定休日をカード形式で表示
   - 電話番号を強調
   - アクセス情報（駐車場、最寄り駅）

4. **SPACE**
   - 店内の雰囲気を写真で紹介
   - 4枚の写真をグリッドレイアウト

5. **ACCESS**
   - 住所を明記
   - Google Map 埋め込み
   - 駐車場情報とアクセス方法

6. **FOOTER**
   - 店名・電話番号・住所を表示

### 技術仕様

- **Pure HTML/CSS/JavaScript**: フレームワーク不使用
- **レスポンシブ**: SP最優先、ブレークポイント880pxでタブレット/PC表示に切り替え
- **SEO対策**: 適切な見出し構造（h1〜h3）、構造化データ（JSON-LD）、メタタグ設定
- **アクセシビリティ**: aria-label、alt属性、スキップリンク対応
- **アニメーション**: IntersectionObserver を使用したフェードイン（上品で最小限）
- **パフォーマンス**: 画像のlazy loading、prefers-reduced-motion対応

### 画像の差し替え方法

Google Driveの画像を使用する場合、以下のIDを実際のファイルIDに置き換えてください：

- `HERO_IMAGE_ID`: HEROセクションのメイン画像
- `SUB_IMAGE_01_ID`: CONCEPTセクションの画像
- `SUB_IMAGE_02_ID` 〜 `SUB_IMAGE_05_ID`: SPACEセクションのギャラリー画像

Google Driveの画像URL形式：
```
https://drive.google.com/uc?export=view&id=【ファイルID】
```

### カスタマイズ

CSS変数（`:root`）で色やスペーシングを一元管理しています。必要に応じて `styles.css` の変数を変更してください。
